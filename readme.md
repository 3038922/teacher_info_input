## 自用命令记录
### pyinstaller配置说明
- 生成一大堆 `pyinstaller main.py`
- 生成单exe `pyinstaller -F main.py`
- 生成带图标文件的EXE `pyinstaller -F -i ./pic/机器人标.ico main.py`
